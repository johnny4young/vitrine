# Bundled fonts

Vitrine bundles these monospaced fonts (popular among developers) so the rendered
output looks identical on any Mac. All are redistributable under open licenses.

| Font | Author | License |
| --- | --- | --- |
| JetBrains Mono | JetBrains | SIL Open Font License 1.1 |
| Fira Code | Nikita Prokopov & contributors | SIL Open Font License 1.1 |
| Hack | Source Foundry | Hack Open Font License (MIT + Bitstream Vera) |
| IBM Plex Mono | IBM | SIL Open Font License 1.1 |
| Roboto Mono | Christian Robertson (Google) | Apache License 2.0 |
| Space Mono | Colophon Foundry (Google) | SIL Open Font License 1.1 |
| Ubuntu Mono | Dalton Maag (Canonical) | Ubuntu Font License 1.0 |
| Geist Mono | Vercel | SIL Open Font License 1.1 |

System monospaced fonts (SF Mono, Menlo, Monaco) are offered in the picker but not
bundled — they ship with macOS.

Full license texts are available from each project's repository. These fonts are
distributed unmodified.
